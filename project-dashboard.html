OnMed Healthcare KPI Dashboard

What I Built: A Power BI dashboard consolidating operational and clinical data for healthcare management decision-making.

Key Skills Demonstrated

  • Power BI dashboard development (data modeling to visualization)
  • SQL optimization (Stored Procedures for performance)
  • Power Automate integration (self-service refresh button)
  • Healthcare data & HIPAA compliance
  • Stakeholder collaboration & requirements gathering

What I Delivered

Business Problem:

Management needed real-time visibility into operations to make staffing and resource decisions.

Technical Solution:

  • Built SQL Stored Procedure combining SQL Server + Postgres EMR data for optimized performance.
  • Created Power BI dashboard with DAX measures for key metrics.
  • Added Power Automate refresh button so executives control data updates.

Dashboard Metrics:

  • Patient volume trends (new vs. returning, ratings)
  • Clinical patterns (visit reasons, prescriptions)
  • Operational insights (peak hours, station utilization)

Business Impact:

A single source of truth enabling data-driven decisions on staffing, resources, and patient experience.
